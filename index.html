   - **Option 1: Embed the QR Code Image**:

     - Upload the downloaded QR code image to your hosting service in the same directory as your `index.html` (e.g., create an `images` folder and upload `qr-code.png`).

     - Modify the `<img>` tag in the footer to reference the image:

       ```html

       <img src="images/qr-code.png" alt="QR code to event page">

       ```

     - Convert your QR code image to a base64 string using a tool like [Base64 Image Encoder](https://www.base64-image.de).

     - Replace the `<img>` tag’s `src` in the footer with the base64 string, like in the provided HTML:

       ```html

       <img src="data:image/png;base64,[your-base64-string]" alt="QR code to event page">

       ```

     - This avoids hosting an external image file but increases the HTML file size slightly.


4. **Test the QR Code**:

   - Save the updated `index.html` and upload it to your hosting service.

   - Open the website in a browser to ensure the QR code displays correctly.

   - Scan the QR code with a smartphone to verify it directs to the correct URL.

   - Ensure the QR code is large enough (at least 1–2 inches in print) for easy scanning.
